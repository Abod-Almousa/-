# Azkary

تطبيق أذكاري يحتوي على أذكار المسلم اليومية مقسمة على عدة أقسام لتسهيل الإستخدام. بالإضافة لمسبحة إلكترونية

## Technologies Used

* Java Programming Language.
* XML Markup Language.
* SQLite database
* Night and Light mode
